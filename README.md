# Nội dung thực hiện
Phần 1: Quản trị Cơ sở dữ liệu (SQL)
Thực hiện tạo bảng và truy vấn các yêu cầu sau:

Liệt kê sinh viên thuộc lớp DI1296A1 và DI1296A2.

Liệt kê sinh viên học môn CT178.

Liệt kê sinh viên lớp CP1296C1 học môn CT178.

Liệt kê sinh viên lớp CP1296C1 học môn CT178 có kết quả học tập đạt loại C.

Phần 2: Đánh giá thông tin (SQL vs NoSQL)
Sử dụng bài kiểm tra CRAAP để đánh giá 4-6 nguồn tài liệu so sánh giữa SQL và NoSQL dựa trên:


C (Currency): Tính thời sự của thông tin.


R (Relevance): Tính liên quan đến ngữ cảnh IT.


A (Authority): Uy quyền của tác giả/tổ chức.


A (Accuracy): Tính chính xác và minh chứng kỹ thuật.


P (Purpose): Mục đích của bài viết (trung lập hay quảng cáo).
